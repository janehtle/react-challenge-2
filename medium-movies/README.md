# Movie Search - MEDIUM
Challenge: Program a movie search app with React hooks and React Router. Utilize an API to fetch movie data based on user search queries. Display search results and implement a detailed view for each movie.


# Instructions
1. Clone the repo
2. Cd into each or the folder you would like to see
3. Install all dependencies/npm necessary using 'npm install'
4. To see it run locally, in your terminal, run 'npm run dev'
